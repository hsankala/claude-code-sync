# Context7 MCP Setup

Adds the Context7 MCP server to the current Claude Code project with local (project-scoped) configuration.

Context7 is an MCP server by Upstash that fetches **up-to-date, version-specific documentation** from library sources and injects it into Claude Code's context. It prevents hallucinated APIs and outdated code patterns. Once configured, append `use context7` to any prompt and Claude Code will pull current docs before generating code.

---

## Pre-flight — Check if Context7 is already configured

Before doing anything else, run:

```bash
claude mcp get context7
```

If Context7 is **already configured**, tell the user clearly and ask:

> ⚠️ Context7 MCP is already configured for this project.
>
> Do you want to remove it and re-add with a new key, or leave it as-is?

If they want to reconfigure, remove it first:

```bash
claude mcp remove context7
```

If it is **not configured**, proceed to Step 1.

---

## Step 1 — Get your Context7 API key

Present this to the user clearly:

> 🔑 **You'll need a Context7 API key to continue.**
>
> ► Log into your Context7 dashboard: **https://context7.com/dashboard**
>
> ► Context7 keys are created per application — create a **new key** for this project.
>
> ► Once created, **save the key to your Google Drive credential spreadsheet** before pasting it here.
>
> ► Paste the API key here when you're ready.

Do not proceed until the user provides the key.

---

## Step 2 — Add Context7 MCP

```bash
claude mcp add --scope local --transport http context7 https://mcp.context7.com/mcp \
  --header "CONTEXT7_API_KEY: <API_KEY>"
```

Substitute the user's actual API key.

---

## Step 3 — Verify

```bash
claude mcp get context7
```

Confirm the output shows:
- **Scope**: Local config
- **Type**: http

If it fails, show the user the full command with their key filled in so they can run it manually and see the raw error.

## Troubleshooting

| Issue | Fix |
|---|---|
| `context7` not found in `claude mcp list` | Re-run the add command — check for typos in the API key |
| Status shows disconnected | Verify the API key is valid at context7.com/dashboard |
| Want to remove it | `claude mcp remove context7` from the project root |

---

## Notes

- The MCP config is stored in `~/.claude.json` — do not edit this file manually
- **Always use `--scope local`** — applies only to the current project directory
- `--scope user` would activate Context7 globally across every project — do not use unless the user explicitly requests it and understands the implication
- Context7 is free and open source (MIT). API keys give higher rate limits and are created per application
